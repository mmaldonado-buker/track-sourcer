Conectar Google Sheets

Configura el backend una sola vez

1
Crea el Google Sheet
Ve a sheets.google.com, crea un nuevo documento y nómbralo SourcerTrack DB. No necesitas crear pestañas — el script lo hace automáticamente.
2
Instala el Apps Script
En el Sheet: Extensiones → Apps Script. Borra el código existente, pega el contenido del archivo Code.gs y guarda con Ctrl+S.
3
Despliega como Web App
Clic en Implementar → Nueva implementación → Aplicación web.
• Ejecutar como: Yo
• Quién tiene acceso: Cualquier persona
Copia la URL que aparece al final.
4
Pega la URL aquí
💡 Tip: También puedes usar la versión offline (datos en localStorage) mientras configuras el Sheet.
